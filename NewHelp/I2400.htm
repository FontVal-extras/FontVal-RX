Message: I2400

prep: Validation for the prep table is not directly implemented, but rasterization testing will validate any executed hinting instructions

The prep table, containing the hinting instructions for the glyphs, is not validated by Font Validator, except to the extent that run-time errors cause glyph validation to fail. To perform detailed checks of hinting instructions, select rasterization testing in the Font Validator, or use a tool such as Microsoft's Flint or VTT.

For more information, see the prep table specification.
